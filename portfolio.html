responsive website

Main goal of this project was to create a website in Bauhaus style, which would be responsive to different viewports and devices.

To start with, the analysis of Bauhaus design main features was conducted, providing some information about the style and combining mood board to visually represent the style. Later on, both prototypes – for mobile and web version were made to make the style in line and check the usability as well as make. Responsive website was designed using two column grid layout. The website has two breakpoints - 30em and 50em. Burger menu was applied for mobile version.

interactive teaser

The goal of this project was by working in a group to deliver an interactive teaser of “Young Frankenstein” movie in such a manner, that teenagers would be interested in watching the movie. Therefore the storytelling, dramaturgy, interactivity, graphical elements and animations created support this goal.

In addition, several user tests with prototypes and pre-final version of product were conducted to adjust the product in line with target audience’s preferences. All the concept was integrated into an interactive production using html, CSS and JavaScript. The website is responsive.

ux/ui

The purpose of this project was to create and develop a relevant digital business concept from the profound ideation to the final launch of a digital start-up company. The idea of the application is to provide necessary information for everyone who is experienced in hunting Northern Lights or planning to travel for the first time to see Northern Lights and create a community.

Market researches, competitors’ analysis and target groups’ surveys were conducted in order to come up with functional, useful application. Customer journey, communication plan, user tests led to even more precise results. To create the very best user experience, following values were priority: useful, usable, findable, credible, desirable, accessible, valuable.
